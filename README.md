# My Shelf

My Shelf on Flutterilla kehitetty mobiilisovellus oman kirjakokoelman hallintaan.

Sovelluksen päätavoitteena on tehdä fyysisen kirjahyllyn hallinnasta helppoa: kirjat voidaan lisätä viivakoodia skannaamalla tai käsin, järjestellä virtuaaliseen kirjahyllyyn ja tallentaa pysyvästi laitteelle.

## Nykyiset ominaisuudet

### Kirjojen lisääminen

* ISBN-viivakoodin skannaus kameralla
* ISBN:n käsinsyöttö
* Google Books -haku
* Open Library -varahaku
* Manuaalinen kirjan lisäys

### Virtuaalinen kirjahylly

* Kirjat esitetään selkämyksinä
* Automaattinen hyllyrivien muodostus
* Drag & Drop -järjestely
* Kirjojen järjestyksen tallennus

### Kirjan hallinta

* Kirjan tietonäkymä
* Kirjan muokkaaminen
* Kirjan poistaminen
* Kansikuvan näyttäminen (jos saatavilla)

### Tallennus

* Kirjat tallennetaan paikallisesti laitteelle
* Kirjajärjestys säilyy sovelluksen uudelleenkäynnistyksen jälkeen

## Teknologiat

* Flutter
* Dart
* Google Books API
* Open Library API
* mobile_scanner
* shared_preferences
