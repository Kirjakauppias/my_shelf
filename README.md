# My Shelf

My Shelf on Flutterilla toteutettu Android-sovellus oman kirjakokoelman hallintaan.

## MVP

- ISBN-viivakoodin skannaus
- Google Books -haku
- Open Library -varahaku
- Kirjan lisääminen kirjahyllyyn
- Virtuaalinen kirjahylly
- Kirjojen järjestely Drag & Dropilla
- Paikallinen tallennus

## Tulevat ominaisuudet

- Manuaalinen kirjan lisäys
- Useita kirjahyllyjä
- Realistiset selkämykset
- Pilvivarmistus
